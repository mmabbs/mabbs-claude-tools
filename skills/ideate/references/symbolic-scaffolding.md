# Symbolic Scaffolding ("Supreme Mathematics")

**Core operation:** Map concepts to a progression structure that provides meaning, sequence, and completion signals.

## Why It Works

Humans naturally chunk information into meaningful sequences. Explicit scaffolding externalizes that, making implicit structure discussable.

## The Pattern

A numbered or staged framework where each position has semantic weight:

- **Position implies sequence** — What comes before/after
- **Position implies relationship** — How elements relate
- **Position implies completeness** — When the cycle closes

## Creating a Scaffold

1. **Identify the progression** — What moves from beginning to end?
2. **Name the stages** — Give each position meaningful labels
3. **Define transitions** — What moves you from stage to stage?
4. **Mark completion** — How do you know you're done?

## Applications

| Context | Scaffolding Approach |
|---------|---------------------|
| **Project phases** | Define stages with names that carry meaning beyond "Phase 1, 2, 3" |
| **Learning progressions** | Map skill development to stages with qualitative character |
| **Rituals and habits** | Daily/weekly/monthly practices mapped to different functions |
| **Decision frameworks** | Numbered criteria where order implies priority |

## Example Scaffolds

**Learning:** Unconscious Incompetence → Conscious Incompetence → Conscious Competence → Unconscious Competence

**Grief:** Denial → Anger → Bargaining → Depression → Acceptance

**Design:** Discover → Define → Develop → Deliver

**Change:** Unfreeze → Change → Refreeze

## Building Your Own

The specific system (Supreme Mathematics, DIKW pyramid, Maslow's hierarchy, etc.) matters less than having *a* symbolic structure you can reason with.

**Questions to ask:**
- What's the natural progression here?
- What happens at each stage?
- What triggers the transition?
- What does "complete" look like?

## Red Flags You Need This

- "This feels like a blob with no structure"
- Can't explain where you are in a process
- No sense of progress or completion
- Stages are implicit but unnamed
